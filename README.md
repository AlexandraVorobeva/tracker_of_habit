-Python <br>
-Django<br>

## Installation
### Clone the repo:<br>

$ git clone https://github.com/SparklingAcidity/tracker_of_habit <br>
$ cd tracker_of_habit <br>

### Docker run:
$ docker-compose up -d --build<br>
